🛠️ C 语言底层逻辑探索 (C-Programming-Practice)
这个仓库是我编程之路的起点，主要记录了在 Visual Studio 环境下练习的各种 C 语言 Demo。通过这些底层练习，我建立起了对内存、算法和计算机逻辑的初步认知。
💻 开发环境
IDE: Visual Studio 2022 (MSVC 编译器)
语言标准: C11 / C99
📂 练习模块说明
这个仓库包含以下核心 Demo 类型：
📁 Basics/: 变量、数据类型、运算符及流程控制语句（if-else, switch, loops）。
📁 Functions/: 函数的定义、调用以及参数传递。
📁 Arrays_Strings/: 一维/多维数组的处理，以及 C 风格字符串的操作。
📁 Pointers/: (核心练习) 指针的基本概念、地址运算。
📁 Structs/: 结构体定义，尝试初步构建简单的数据模型。
🌟 核心练习案例
经典算法 Demo: 包含冒泡排序、二分查找等基础算法的 C 语言实现。
逻辑处理: 练习了复杂的嵌套循环和条件判断，增强逻辑思维。
内存初探: 通过指针练习，理解数据在内存中的存储形式。
📝 学习笔记
关于环境: 在 Visual Studio 中开发时，注意了 scanf_s 与标准 scanf 的区别，以及如何配置项目属性。
心得: C 语言虽然“硬核”，但它让我明白了 Python 里的 list 在底层是如何运作的。这种“向下扎根”的过程对我的数据分析之路大有裨益。
📅 维护状态
目前重心已转向 Python 数据分析，本仓库将作为底层知识库进行不定期维护和查阅。
